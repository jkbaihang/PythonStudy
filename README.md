### list,tuple
- **list**:可变数组`classmates = ['Michael', 'Bob', 'Tracy']`
 ```
 classmates[-1] #末尾元素
 classmates.pop() #删除末尾元素
 classmates.pop(1) #删除索引1元素
 classmates[1] = "Chen"  ##替代元素
 classmates.insert(1,'Jack')
 classmates.append('Bai')
 ```
- **tuple**:不可变数组`classmates = ('Michael', 'Bob', 'Tracy')`
